# Add Your Background Images Here

## Quick Start
1. Download backgrounds from Freepik.com
2. Rename them to: bg1.jpg, bg2.jpg, bg3.jpg, bg4.jpg
3. Place them in this folder
4. Refresh your login page - backgrounds will auto-rotate every 10 seconds!

## Suggested Freepik Search Terms
- "workspace background"
- "office desk minimal"
- "gradient abstract background"
- "professional background"
- "tech background"

## Image Requirements
- Format: JPG or PNG
- Size: 1920x1080 or higher
- File size: Under 500KB (optimized)
- Style: Professional, clean

See BACKGROUND_SETUP_GUIDE.md in root folder for detailed instructions.
